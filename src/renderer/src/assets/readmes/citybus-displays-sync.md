# citybusDisplaysSync

**by StanleyDudek** · [GitHub](https://github.com/StanleyDudek/citybusDisplaysSync)

Fixes display bugs in BeamNG.drive's citybus displays and busline UI app, and **syncs bus destination boards across all players** in BeamMP — so everyone sees the correct route and stop information in real time.

## What it does

- Corrects visual glitches in the stock citybus front/side/rear destination displays
- Keeps the busline UI app consistent between the driver and all passengers/observers in a BeamMP session
- Lightweight client-side mod — no server plugin required

## Recommended with

- [CareerMP](https://github.com/StanleyDudek/CareerMP) — Online career mode for BeamMP
- [Buber](https://github.com/DeadEndReece/Buber) — Bus route ride-share gameplay for CareerMP
- [freeRoamMP](https://github.com/StanleyDudek/freeRoamMP) — Free roam multiplayer mode

## Installation

This is a **client mod** distributed via `Resources/Client/`. The Content Manager installs it automatically — players connecting to your server will receive it.
